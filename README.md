# 🏏 IPL Data Analysis Dashboard (2008–2024)

## 📌 Project Overview
This project is an interactive **IPL Data Analysis Dashboard** built using Power BI.  
It provides insights into team performance, player statistics, and match outcomes across IPL seasons (2008–2024).

## 🔍 Key Insights

- Mumbai Indians have the highest overall wins in IPL history  
- Gujarat Titans show one of the best win percentages  
- Virat Kohli leads in batting performance  
- Yuzvendra Chahal dominates bowling statistics  
- AB de Villiers holds the record for most *Man of the Match awards (25 times)*  
- Toss-winning teams win approximately **51%** of matches  

---

## 📈 Features

- Dynamic Points Table with Net Run Rate (NRR)  
- Team-wise Win Percentage Analysis  
- Orange Cap (Top Batsman) & Purple Cap (Top Bowler)  
- Batting & Bowling Strike Rate Analysis  
- Interactive filters (season-wise exploration)  

---

## 🛠 Tools & Technologies
